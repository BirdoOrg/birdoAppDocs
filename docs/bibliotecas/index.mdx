O *BirdoApp* fornece duas bibliotecas diferentes para organizar os _Assets_ da produção. Uma para organizar os _RIGs_ e outra para organizar bancos de reaproveitamento para cada versão desses _RIGs_. 

:::warning
   Para uso das Bibliotecas do BirdoApp, é necessário estruturar os RIGs do projeto seguindo os critérios descritos em ["Estrutura de RIG"](./rig.md)
:::
